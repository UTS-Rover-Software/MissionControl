## Welcome to Mission Control

Mission Control is the comprehensive documentation site for the UTS Rover Team's rover control and visualization system. This site provides guides, references, and resources for setting up, using, and contributing to the project.

## What's Included

- **Setup Guides**: Step-by-step instructions for frontend and backend setup.
- **User Manual**: How to operate and interact with the rover control interface.
- **Widget Library**: Detailed documentation for all available widgets.
- **Architecture Overview**: Insights into the frontend and backend design.
- **Contributing Guidelines**: How to get involved and follow best practices.

## Getting Started

If you're new to the project, start with the [Setup](/general/setup) section to get your environment ready. Then, check out the [Manual](/general/manual) for usage instructions.

For developers, explore the [Frontend](/frontend/) and [Backend](/backend/routers) sections to understand the codebase.
