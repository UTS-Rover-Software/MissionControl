# Widgets

Overview of the widgets available in the Mission Control application. Widgets provide specialized interfaces for rover control, visualization, and monitoring.

## Available Widgets

- [**Camera Feed**](camera-feed.md): Live video streams from multiple cameras
- [**Science Widget**](science-widget.md): Controls for science module operations
- [**Bounding Box for Widgets**](bounding-box-for-widgets.md): Multi-screen layout management
- [**Pointcloud**](pointcloud.md): 3D depth visualisation from RGB-D cameras
- [**Nav2 Target Queue**](nav2-target-queue.md): Autonomous navigation target list
- [**RVIZ**](rviz.md): Robot pose and situational awareness
- [**2D Aerial Map SLAM**](2d-aerial-map-slam.md): RTAB-Map generated aerial maps
- [**VESC Info Readonly**](vesc-info-readonly.md): Speed controller status
- [**Machine Usage Readonly**](machine-usage-readonly.md): Jetson system stats
- [**Rosout Console**](rosout-console.md): ROS log display
