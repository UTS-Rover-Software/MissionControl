# 2D Aerial Map SLAM

## Description

Displays 2D aerial map generated by RTAB-Map SLAM. Shows occupancy grid with robot position marker.

Task specific: Yes (autonomous)  
Guaranteed to be used: Maybe (2.5/5 importance)

## Implementation

### Data Sources
- Subscribe to /rtabmap/grid_map (occupancy grid)
- Optionally /rtabmap/proj_map (lidar style)

### Visualization
- 2D map rendering
- Robot position as "X" marker
- Placard locations from JSON

### Technologies
- ROS2, Canvas/SVG rendering
